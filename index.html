Botões


Estilos de botões


Botão Preenchido:

São os botões usados para as ações principais do usuário, são os que possuem maior ênfase.

Botão Contorno:

São usados como uma ação alternativa ao botão principal, possuem uma ênfase média.

Botão Texto:

São usados com a menor prioridade, geralmente quando há multiplas opções, possue uma ênfase baixa.

Botões Alternativos

Usados para dar contraste em fundo com a cor azul correios. As versões contorno e texto não devem ser usadas em fundo claro, pois não geram contraste suficiente para garantir uma boa leitura.
